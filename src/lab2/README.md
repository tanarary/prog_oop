# Лабораторная работа-2 — Коллекция объектов (Python 3.x)

## Цель работы

* Научиться работать с коллекциями объектов.
* Понять разницу между моделью сущности и контейнером объектов.
* Реализовать собственный контейнерный класс.
* Освоить итерацию по объектам.
* Реализовать базовые операции управления коллекцией.

## Вариант 10. Медицина

### Класс:
* Patient - пациент
* Класс коллекции - PatientRegistry
 

## Краткое описание класса

PatientRegistry — это контейнерный класс для управления коллекцией пациентов. Внутри хранит список объектов Patient.

## Методы коллекции

### Базовые операции

| Метод | Описание |
|-------|----------|
| `add(patient)` | 	Добавить пациента в коллекцию |
| `remove(patient)` | Удалить пациента из коллекции |
| `remove_at(index)` | Удалить по индексу |
| `get_all()` | Получить список всех пациентов |
| `clear()` | Очистить коллекцию |

### Поиск

| Метод | Описание |
|-------|----------|
| `find_by_id(patient_id)` | Поиск пациента по ID |


### Магические методы

| Метод | Описание |
|-------|----------|
| `__len__` | Получение количества пациентов |
| `__iter__` | Итерация по коллекции |
| `__getitem__` | Доступ по индексу |
| `__str__` | Строковое представление |
| `__contains__` | Проверка наличия (patient in registry) |

### Сортировка

| Метод | Описание |
|-------|----------|
| `sort_by_name(reverse))` | Сортировка по ФИО |

### Фильтрация (возвращают новую коллекцию)

| Метод | Описание |
|-------|----------|
| `get_active()` | Получить пациентов на лечении |
