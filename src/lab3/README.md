## Вариант 10. Медицина

### Класс:
* Patient - пациент
 
**Атрибуты:**
- _full_name — ФИО пациента
- _age — возраст
- _blood_type — группа крови
- _diagnosis — диагноз
- _temperature — температура тела
- _status — статус (на лечении/выписан/направлен)
- _admission_date — дата поступления
- _discharge_date — дата выписки
- _patient_id — уникальный идентификатор

**Методы:**
- discharge() — выписать пациента
- refer_to_specialist(specialist) — направить к специалисту
- has_fever() — проверка лихорадки
- is_temperature_normal() — проверка нормальной температуры
- can_donate_blood() — проверка возможности донорства
- get_health_status() — получить статус здоровья
- calculate_treatment_cost() — расчёт стоимости лечения (полиморфный)
- get_patient_category() — возвращает категорию пациента (общий интерфейс)
- __str__() — строковое представление

### Дочерний класс `EmergencyPatient` (Экстренный пациент)

**Новые атрибуты:**
- _severity — степень тяжести (1-5)

**Новые методы:**
- requires_resuscitation() — требуется ли реанимация (возвращает True при severity ≥ 4)

**Свойства (property):**
- severity — возвращает степень тяжести

**Переопределённые методы:**
- __str__() — добавляет к выводу маркер экстренного пациента и степень тяжести

### Дочерний класс `ChronicPatient` (Хронический пациент)

**Новые атрибуты:**
- _chronic_disease — хроническое заболевание
- _years_with_disease — сколько лет болеет

**Новые методы:**
- is_in_exacerbation() — проверка обострения (температура ≥ 38.0°C)

**Свойства (property):**
- chronic_disease — возвращает название заболевания
- years_with_disease — возвращает стаж болезни

**Переопределённые методы:**
- __str__() — добавляет к выводу маркер хронического пациента (или обострения) и стаж болезни


### Различия между классами

| Класс | Особенность | Что отличает | 
|-------|--------|--------|
| Patient | обычный пациент| базовая логика | 
| EmergencyPatientt |экстренный пациент | есть заболевание и стаж, можно проверить обострение | 
| ChronicPatient | хронический пациент | есть заболевание и стаж, можно проверить обострение|

ChronicPatient: добавляет ⚠️ ОБОСТРЕНИЕ или 📋 ХРОНИЧЕСКИЙ и стаж болезни

выделение только экстренных пациентов
выделение только хронических пациентов

has_fever() — проверка лихорадки

get_health_status() — получение статуса здоровья
can_donate_blood() — проверка возможности донорства

refer_to_specialist() — направление к специалисту
